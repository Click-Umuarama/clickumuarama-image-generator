## Instalação

- Clone o projeto e entre em seu diretório:

```console
$ git clone git@github.com:agenciaecode/clickumuarama-image-generator.git
```

## Pré-requisitos

- [Node](https://nodejs.org/en/)
- [pnpm](https://pnpm.io/) (opcional)

## Configuração inicial

- Instalar dependências da aplicação:

```console
$ pnpm install
```

- Inicializar aplicação:

```console
$ pnpm run dev
```

## Scripts Disponíveis
